# redis常用配置

|  配置项  |  示例  | 说明   |
|----|----|----|
|  daemonize  |  daemonize yes  |  是否启用后台运行，默认no  |
|  port  | port 6379   | 设置端口号   |
|  logfile  |  logfile  | 设置日志文件   |
|  databases  |  databases 255  |  设置数据库总量  |
|  dir  |  dir 数据文件目录  |  设置数据文件存储目录  |
|  requirepass  |  requirepass 密码  | 设置密码   |
